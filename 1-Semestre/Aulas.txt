    MINHAS ANOTAÇÕES:
        Terminal do switch:
            CTRL + SHIFT + 6: Para de executar uma ação no terminal do swtich.
            enable: ativa o modo root(modo de admin)
            disable: volta para o modo exec
            configure terminal: abre a aba de config
                hostname "nome": cria um novo nome ao inves do padrao switch
                line console 0: abre dentro de config a linha do console 0
                    password "senha": seta a senha
                        login: coloca a senha no login
                    exit: sai do line e volta um
                line vty 0 15: abre dentro de config a linha dos 16
                    password "senha": seta a senha
                        login: coloca a senha no login
                    exit: sai do line e volta um
                service password-encryption: cria a cifra do texto plano da senha na onfiguração geral
                banner motd @: deixa criar um banner até digitar outro @ no final para terminar
                vlan "numero dela": cria a vlan
                    name: coloca um nome para a vlan
                interface fastEthernet 0/1: acessa as portas, no caso a 0/1
                    switchport mode access: acessa o modo de acesso da porta
                        switchport access vlan 20: associa a porta a vlan, no caso a 20
                int gi 0/1: acessa a porta do gb
                    sw mo trunk: transforma a porta em trunk
                do sh vl br: mostra o brief
            show vlan brief: mostra as vlans
            show running-config: mostra toda a configuração
            write: salva
        Uma porta trunk permite a passagem entre os switchs
        Podem ser criadas vlans do numero 2 até 999
        Passos:
            1- Criar vlans
            2- Verificar as vlans criadas
            3- Associar as portas as vlans
            4(opcional)- Configurar portas livres

        VTP = 1 servidor e n clientes
            o vtp server é onde criamos todas as vlans
            os outros switchs são clientes, onde eles pegam as vlans do server
            configuração server:
                enable
                conf t
                vtp mode server
                vtp domain teste.com
                vtp password teste123
                vtp version 2
                exit
                wr
            configuração clientes:
                enable
                conf t
                vtp mode client
                vtp domain teste.com
                vtp password teste123
                exit
                wr
        STP 
            impede que o switch entre em um loop infinito
            comandos:
                enable
                conf t
                spanning-tree vlan 1 --> habilito
                spanning-tree mode rapid-pvst --> habilito uma velocidade maior 
                no spanning-tree vlan 1 --> desabilito
        Port Security
            limtar a quantidade de equipamentos que pode acessar a rede em um único ponto
            comandos:
                enable
                conf t
                int fa 0/1
                sw mode access
                sw port-security
                sw port-security maximum 2 --> deix ano máximo dois computadores nessa porta
                    colocar ? depois de port-security vai dar as opções possíveis
                sw port-security violation shutdown --> ele desliga(bloqueia) a porta
                    colocar ? depois de violation vai dar as opções possíveis
                sw port-security mac-address sticky
        Eterchannel
            agregação de link --> aumenta a velodade da rede, juntando a velocidade dos cabos
            comandos:
                enable
                conf t
                int range fa 0/1-4
                sw mode trunk
                channel-group 1 mode active
                exit
                int port-channel 1
                sw mode trunk
                exit
                exit
